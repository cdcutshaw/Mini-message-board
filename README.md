This project is part of The Odin Project's full stack Javascript curriculum. link to The Odin Project page: https://www.theodinproject.com/lessons/node-path-nodejs-mini-message-board

The Mini Message Board is a simple Node.js and Express application that demonstrates basic concepts of routing, templating, and dynamic data rendering. Users can view a list of messages, add new messages via a form, and view the details of individual messages. The project highlights foundational Express.js features such as setting up routes, handling form submissions, and rendering views with EJS.
